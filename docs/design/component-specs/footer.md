# Component Spec — Footer

| 項目 | 値 |
|------|-----|
| 名称 | Footer |
| ベース | — |
| 配置 | 全ページ最下部 |
| 対応 AC | AC-A2-4 |

## 用途
法務・サポートリンクとサイトマップ的導線を提供。

## 構造
- ロゴ + 簡潔なタグライン
- リンク群: 利用規約 / プライバシーポリシー / 特定商取引法表記 / FAQ / お問い合わせ
- 主要導線: 占い師を探す / 鑑定を探す / ブログ / 占い師として登録
- 占術カテゴリリンク（任意, SEO 内部リンク）
- コピーライト

## スタイル
- 背景: brand-navy（ダーク面）、テキスト白（15.62:1）/ 補助 gray-300（10.60:1）
- リンクホバー: brand-orange-light（テキスト可 on navy）

## レスポンシブ
- base: 縦積みリンク群 / md: 2列 / lg+: 多列（4〜5列）

## アクセシビリティ
- `<footer>` ランドマーク、`<nav aria-label="フッターナビゲーション">`
- リンク識別可能、フォーカスリング（navy 上は white/orange リング）、44px

## 禁止
- グロー、過剰シャドウ、白文字 on 暖色塗りの小サイズ
